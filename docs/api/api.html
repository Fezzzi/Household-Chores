Load

Resources

GET​/resources​/{resource}
Returns information for given resource

Auth

POST​/auth​/sign-up
Signs up the user
PUT​/auth​/log-in
Logs in the user
PUT​/auth​/reset
Resets user's password
DELETE​/auth​/delete
Deletes user's account

Connections

GET​/connections​/find
Searches for users with names/emails matching query provided
POST​/connections​/request
Sends connection request to the user
PUT​/connections​/{action}
Performs given connection action
DELETE​/connections​/{action}
Performs given connection action

Households

GET​/households​/load
Loads information about user's households
POST​/households​/create
Creates new household
DELETE​/households​/{action}
Deletes/Leaves given household
PUT​/households​/approve
Approves household invitation
DELETE​/households​/ignore
Ignores household invitation

Settings

GET​/settings
Fetches data for a settings page
PUT​/settings
Updated data for a settings page
PUT​/dialogs​/disable
Disables appearing of specified dialog
